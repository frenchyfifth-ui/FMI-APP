# Human Layer

Human viability includes population scale, skill diversity, cultural adaptability, governance capacity, and social cohesion. Psychological resilience is treated as a core resource.

## Operational Constraints
1.  **Identity is not Logic:** Identity is treated as a narrative layer, not a control layer.
2.  **Biological Maintenance:** Sleep, movement, and nutrition are "hard" dependencies.
3.  **Social Cohesion:** Trust is measured by transaction reliability (showing up), not shared belief.

## Integration
An applied example of constraint-first human coordination is provided in `99_appendix/integrated_family_field_protocol.md`.

This protocol demonstrates how social cohesion, execution discipline, and psychological regulation can be maintained under sustained load without reliance on complex external systems.
