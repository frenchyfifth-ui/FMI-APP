# JOURNEY_LOGS

## Purpose
A chronological, immutable record of decisions, artifacts, and experiments.

## Format
Files are named: `YYYY-MM-DD_<artifact_slug>.md`

## Content Rules
1.  **Append-Only:** Do not rewrite history.
2.  **Context:** What triggered this action?
3.  **Output:** Link to the specific Artifact produced.
4.  **Next:** What is the single next executable step?

## Integration
The `JOURNEY_LOGGER_AGENT` automates the creation of these logs.
