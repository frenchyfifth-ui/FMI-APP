# Scope and Limits

This hypothesis is limited to **conceptual modeling** and **execution runtime**.

## Excluded
- Political advocacy
- Emergency response planning
- Speculative or unverified technologies
- "Life OS" self-help branding
- Guru-based advice

## Temporal Scope
Generational (20–100 years). This is not immediate crisis response guidance.

## Operational Scope
The **FMI Execution Kernel** is strictly a runtime for managing high-capacity cognitive output. It does not provide meaning; it protects the ability to create it.
